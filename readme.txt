WordPress同步文章到另外的WordPress，将不同的WordPress联系在一起，不用再在不同的博客之间拷贝来拷贝去了。

== Description ==

传统的wordpress文章同步插件只能实现发布文章时同步到另外一个博客，而本插件可以实现同步到多个博客，而且还能在编辑文章的时候进行同步。

== Installation ==

1、把插件文件夹上传到/wp-content/plugins/目录<br />
2、在后台插件列表中激活它<br />
3、在“插件-WP2WP”菜单中进行同步配置<br />
4、完成后即可
